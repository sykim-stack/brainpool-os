# CoreNull Seed System v1.0 (World Layer)

CoreNull은 삶의 여정을 가치로 전환하는 공간입니다. 본 문서는 씨앗 시스템의 핵심 로직과 세계관을 정의합니다.

## 1. 핵심 개념 및 기술 매핑
| 세계관 | 정의 | 기술 매핑 |
| :--- | :--- | :--- |
| **🌱 씨앗 (Seed)** | 사용자가 스스로에게 한 약속 | `category` (purpose_type = seed) |
| **🌿 성장 (Growth)** | 씨앗방 안의 모든 기록 | `post` (Message type = post) |
| **🌸 꽃 (Flower)** | 씨앗이 현실이 된 순간 | `bloom_date` 도달 시 자동 트리거 |
| **🍎 열매 (Fruit)** | 경험이 콘텐츠가 되는 순간 | `post` (Message type = fruit) |
| **📦 창고 (Storage)** | 가치 공유 및 상품 매칭 공간 | 열매 생성 시 자동 오픈 |

## 2. 성장 흐름 (Life Journey)
1. **🌱 씨앗**: 사용자가 심음 (방 생성 및 `bloom_date` 설정)
2. **🌿 성장**: 포스트, 사진, 발자취 기록 (Context 누적)
3. **⏳ 카운트다운**: `bloom_date`를 향해 진행
4. **🌸 꽃**: `bloom_date` 도달 → 시스템 자동 트리거 (Event 발생)
5. **🍎 열매**: 주인이 직접 경험을 콘텐츠화 (수동 진화)
6. **📦 창고**: 열매 분석 기반 랜덤 상품 매칭 및 진열

## 3. 핵심 원칙
- **씨앗 없이는 꽃 없다**: 모든 가치는 여정의 시작에서 비롯됩니다.
- **꽃 없이는 열매 없다**: 현실이 된 순간만이 가치를 맺을 수 있습니다.
- **열매 없이는 창고 없다**: 경험의 공유 없이는 가치의 공간도 열리지 않습니다.
- **창고 없이는 상품 없다**: 여정을 완주한 사람만이 시스템의 혜택을 누립니다.

## 4. 데이터 무결성 (Single Source of Truth)
- **수확 여부**: `harvested_at IS NOT NULL`로 판정합니다.
- **Library View**: 수확된 Fruit만 필터링하여 보여주는 아카이브 뷰입니다.
- **Storage**: 시스템은 열매 콘텐츠를 분석하여 최적의 상품을 매칭합니다.
